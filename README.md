# Automatic-Irrigation-System

A device that measures soil moisture and performs irrigation if the moisture content is below the required value.
**Tools used:**
1. >   submersible water pump
2. >   arduino uno
3. >   transistor (bdx54c)
4. >   soil moisture(Yl-69)
5. >   potentiometer(10k)
